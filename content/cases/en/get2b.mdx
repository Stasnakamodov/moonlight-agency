# Get2B — Next-Gen B2B Marketplace

Get2B is a modern B2B sales platform connecting suppliers and buyers in a single ecosystem.

## Challenge

The client wanted to create a wholesale marketplace with automatic catalog sync from their ERP, AI product recommendations, and real-time notifications.

## Solution

We built a full-stack platform on Next.js with server components, Supabase for database and auth, and ERP integration via REST API.

### Key Features

- Automatic catalog sync from ERP
- AI product recommendations based on order history
- Real-time notifications via WebSocket
- Multi-level discount system
- Admin panel with analytics

## Result

The platform processes over 1,000 orders per month, and AI recommendations increased average order value by 25%.
