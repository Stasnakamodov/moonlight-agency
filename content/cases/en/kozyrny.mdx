# Kozyrny — Crypto Platform

Telegram Mini App for cryptocurrency exchange and management with intuitive interface.

## Challenge

Build a convenient Telegram Mini App for cryptocurrency operations with high security and simple UX.

## Solution

We created a Mini App on React with Telegram WebApp API integration, connected multiple crypto providers, and implemented a verification system.

### Key Features

- Telegram Mini App with native UX
- Multi-currency wallet
- P2P exchange with escrow
- KYC verification
- Push notifications via Telegram Bot

## Result

The app reached top Mini Apps in the finance category, 15K+ active users in the first month.
