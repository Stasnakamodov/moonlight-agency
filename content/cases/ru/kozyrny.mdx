# Козырный — крипто-платформа

Telegram Mini App для обмена и управления криптовалютами с интуитивным интерфейсом.

## Задача

Разработать удобное Telegram Mini App для криптовалютных операций с высоким уровнем безопасности и простым UX.

## Решение

Мы создали Mini App на React с интеграцией Telegram WebApp API, подключили несколько крипто-провайдеров и реализовали систему верификации.

### Ключевые фичи

- Telegram Mini App с нативным UX
- Мультивалютный кошелёк
- P2P обмен с эскроу
- KYC верификация
- Push-уведомления через Telegram Bot

## Результат

Приложение вышло в топ Mini Apps в категории финансы, 15K+ активных пользователей в первый месяц.
