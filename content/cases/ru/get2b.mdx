# Get2B — B2B маркетплейс нового поколения

Get2B — это современная платформа для B2B-продаж, которая объединяет поставщиков и покупателей в единой экосистеме.

## Задача

Клиент хотел создать маркетплейс для оптовых продаж с автоматической синхронизацией каталога из 1С, AI-рекомендациями товаров и real-time уведомлениями.

## Решение

Мы разработали full-stack платформу на Next.js с серверными компонентами, Supabase для базы данных и авторизации, и интеграцией с 1С через REST API.

### Ключевые фичи

- Автоматическая синхронизация каталога из 1С
- AI-рекомендации товаров на основе истории заказов
- Real-time уведомления через WebSocket
- Мультиуровневая система скидок
- Административная панель с аналитикой

## Результат

Платформа обрабатывает более 1000 заказов в месяц, а AI-рекомендации увеличили средний чек на 25%.
