## Советы по использованию промптов

### Модификаторы для стиля

Добавляй в конец промпта по необходимости:

| Стиль | Добавить в промпт |
|-------|-------------------|
| Более реалистично | `photorealistic, studio lighting, 8K` |
| Более плоский / вектор | `flat vector, no shadows, no gradients, clean lines` |
| Больше glow | `ethereal glow, bloom effect, lens flare, light rays` |
| Noise/grain | `film grain texture, subtle noise overlay` |
| Японский минимализм | `wabi-sabi, zen minimalism, negative space` |
| Sci-fi | `futuristic, holographic, HUD interface style` |

### Негативные промпты (Stable Diffusion)

```
Negative prompt: colorful, bright background, white background, gradient rainbow,
cartoon, childish, clip art, low quality, blurry text, watermark, busy,
cluttered, ornate, decorative borders, 3D realistic moon (for flat versions),
lens flare (unless wanted), chromatic aberration
```

### Параметры Midjourney

```
--v 6           (последняя версия)
--style raw     (меньше "улучшений" от MJ, ближе к промпту)
--ar 1:1        (квадрат для лого) | 16:9 (слайды) | 9:16 (stories)
--tile           (для бесшовных паттернов)
--no text       (если MJ добавляет нежелательный текст)
--s 250         (более стилизованно) | --s 50 (ближе к промпту)
```

### Порядок работы

1. Сгенерируй 4-8 вариантов по основным промптам логотипа (раздел 1)
2. Выбери 2-3 направления, доработай промпты
3. Финальный логотип перерисуй в Figma / Illustrator (AI-генерации не годятся для финального лого)
4. Используй AI для фонов, текстур, иллюстраций — это их сильная сторона
5. OG-images и социальные баннеры можно делать полностью через AI + типографику в Figma
